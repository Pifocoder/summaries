# 1 Метод - Manber
Разбиваем документ на подстроки фиксированной длины, потом алгоритм Карпа-Рабина. Сходство - отношение общих подстрок к размеру документа.

# 2 Метод - Broder
Документ разбивался на шинглы и находилось пересечение множеств, используя алгоритм Карпа-Рабина.

# 3 Метод - Словари
Строится два словаря, в которые входят слова со средним количеством встречаемости, потом находится пересечение этих словарей
Высокая эффективность на маленьких документах
Усовершенствование - это создание К словарей и удаление 30% текста для каждого (К = 10)
