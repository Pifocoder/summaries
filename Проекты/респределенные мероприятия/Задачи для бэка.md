# all_events
На страницу должен передаваться массив словарей мероприятий
	Поля
		event_name - название мероприятия
		event_id - id мероприятия
		event_preview - анонс мероприятия

# my_events
На страницу должен передаваться массив словарей мероприятий для определенного пользователя
	Поля
		event_name - название мероприятия
		event_id - id мероприятия
		event_preview - анонс мероприятия

# profile
